## 🧠 Cómo uso IA en mi flujo de trabajo

Integro herramientas de **inteligencia artificial** como parte de mi proceso de desarrollo, no para reemplazar criterio técnico, sino para **acelerar y mejorar el trabajo**.

Uso IA para:

- exploración de soluciones y prototipado

- refactorización y mejora de código  
- documentación técnica  
- generación de pruebas y validación de ideas  
- automatización de tareas repetitivas del desarrollo

Mi enfoque es usar IA como un **copiloto de productividad**, manteniendo siempre revisión, criterio y aprendizaje propio.
